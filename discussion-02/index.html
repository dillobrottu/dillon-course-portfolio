Deconstructing Web Design

Link to the website: https://reverb.com/Links to an external site.

The website I have chosen for this discussion is Reverb, a online music retailer and marketplace.
Reverb uses the CSS Box Model across the entire site, utilizing it to show listings of instruments in rows and columns. On the home page,
it uses the flex box model to show a list of instruments for the user to view.
Each box is equally spaced out from each other and can flex itself to fit on any sized screen.
The model can format itself to appear in different ways, in a horizontal rows, vertical rows, or a combined set of rows.
Each box shows an image, the title, the price, and quality of the listing, all equally spaced out from other listings.
Some listings are sized differently from others, to show a recommended or sponsored listing.
Reverb's use of the box model allows for the display of many different items that are being sold across the world.
